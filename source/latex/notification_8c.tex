\doxysubsection{Detailed Description}
User Notification Definition File. 

\begin{DoxyDate}{Date}
11-\/Dec-\/2018 
\end{DoxyDate}
\begin{DoxyVersion}{Version}
04.\+07.\+01
\end{DoxyVersion}
This file defines empty notification routines to avoid linker errors, Driver expects user to define the notification. The user needs to either remove this file and use their custom notification function or place their code sequence in this file between the provided USER CODE BEGIN and USER CODE END.
